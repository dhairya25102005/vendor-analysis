📊 Vendor Performance Analysis Dashboard
📌 Project Overview
This project analyzes vendor and brand sales performance using SQL, Python (Pandas), and Power BI.
The objective is to evaluate vendor efficiency, identify top-performing vendors and brands, and detect low-performing vendors impacting profitability.

The analysis integrates data cleaning, exploratory data analysis (EDA), SQL-based data transformation, and interactive visualization to generate meaningful business insights for decision-making.
